# Instant Letterhead Maker

A professional, standalone web application designed for schools to generate instant, beautifully formatted letterheads and PDF documents.

## Features
- **Modern UI:** Built with Nithara design language (Glassmorphism & animations).
- **Two-Step Workflow:** 
    1.  **School Setup:** Enter School Name, Code, Address, Place, and Date.
    2.  **Document Creation:** Real-time character-by-character "Paper Preview" of your letter.
- **Smart Letterhead formatting:** 
    - Auto-caps for School Name and Subheadings.
    - Centered and underlined subheadings (Subject/Title).
    - Consistent professional header layout.
- **Portable & Standalone:** Designed to work as a web-view or a separate web app.
- **PDF Generation:** High-quality PDF document creation with `jsPDF`.

## Installation
1.  Clone this repository.
2.  Ensure `jspdf.umd.min.js` is available in the parent folder or update the path in `index.html`.
3.  Open `index.html` in any modern browser.

## Tech Stack
- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- [jsPDF](https://github.com/parallax/jsPDF)
